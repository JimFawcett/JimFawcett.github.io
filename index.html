"Good judgement comes from experience, and experience comes from bad judgement."
- Jim Horning This site provides documentation for JimFawcett repositories, related design notes, and blogs. Most of the repositories contain C++ code. That will be augmented, over time, with C# and Java projects and eventually with some python and node.js code. Goals for development of this site are to:
  • Provide, for former students and other interested developers, easy access to selections of code from my Syracuse University Web Site and new code developed after my retirement from SU (May 2019).
  • Experiment with ways to make reusable fairly large collections of code. To be reusable the code must be simple and reliable and easy to find, understand, and test.
  • Illustrate basic, intermediate, and advanced code structures as clearly as possible.
The plan to move toward those goals has several steps:
  • Create repositories that are, in most cases, relatively small, each with construction tests and many with regression test sequences.
  • For each repository, provide a web page, linked from the repository's Readme.md file, that describes repository content, and eventually provides design documentation using package and class diagrams and descriptions of the responsibilities of each package.
  • Provide design notes, blogs, and annotated lists of resources.
Construction is just starting on this site. For a while you will find some incorrect links and a lot of missing content. Some of the current content was ported from my Syracuse University Website and may not be entirely relevant in this context. I will be modifying or removing that over the next few weeks.
