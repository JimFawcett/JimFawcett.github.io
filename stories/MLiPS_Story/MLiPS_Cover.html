MLiPS Story Prototype

Author: Mike Corley

Story Content:

This is a story about building a Machine Learning framework integrated with PowerShell cmdlets. Each page of the story is linked in the Table of Contents (TOC) dropdown.

Concept:

Processing for machine learning requires:
  • Collecting large data sets for learning
  • Selecting a Machine Learning (ML) algorithm out of many candidates
  • Providing the infrastructure for feeding the data to the ML algorithm
  • Post processing the resulting output to turn its data into information
There are many possible paths through this processing maze. MLiPS provides an infrastructure for assembling an ML pipeline with the help of PowerShell cmdlets, designed for this purpose.
