Starting C++ Story

This is a story about the C++ programming language. It will:
  • Start by providing a synopsis of the language and its environment
  • Chapter 1. - Introduce basics
  • Chapter 2. - Survey C++ classes
  • Chapter 3. - Discuss Inheritance
  • Chapter 4. - Show how Compound Objects are created and used
  • Chapter 5. - Introduce Templates
  • Chapter 6. - Conclude with a survey of the code repositories and how you might start learning C++
You can sequence through the story using "N" and "P" buttons on the right, or use the "TOC" Table Of Contents links. Most of the story pages have navigation menus that allow you to navigate away from story content. To return, simply use the "R" Return button. Any time you want to leave the story, simply use the "E" Exit button.
