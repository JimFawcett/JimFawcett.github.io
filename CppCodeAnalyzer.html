CppCodeAnalyzer  Repository

Uses rule-based parser to build AST holding data for display

Concept:

CppCodeAnalyzer executes static type-based code analysis on specified files in a specified directory tree. It evaluates metrics and provides an Abstract Syntax Tree representation for the analysis results.

Design:

BlogParser.html Describes how the Parser core of CppCodeAnalyzer works.

Build:

CppCodeAnalyzer was built with Visual Studio Community Edition - 2019 and tested on Windows 10.
