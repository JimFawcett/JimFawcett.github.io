CppCodeAnalyzer Repository

CppCodeAnalyzer executes static type-based code analysis on specified files in a specified directory tree. It evaluates metrics and provides an Abstract Syntax Tree representation for the analysis results.

Resources:

CppCodeAnalyzer repository CppCodeAnalyzer was built with Visual Studio Community Edition, 2019. You may have to retarget the solution for the IDE version you are using.
