The prototypes presented here were developed for this site. Several of them are used throughout the site with little or no modification.
You may find them to be good for learning about page structuring and building widgets to improve page ergonomics.
The Bites pages need code documentation similar to that in PhotoSizer bite (that code needs reformatting). Prototypes are Pages, widgets, or compile-able code used to decide how to implement site entities. This track is a place to view prototype pages and widgets developed for this site.
  • Thread and free pages are starting points for new pages for threads and for standalone information.
  • Widgets provide components like resizeable photos, splitter bars, and slide-in panels.
