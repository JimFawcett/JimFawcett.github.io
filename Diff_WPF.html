Diff_WPF Repository

Diff_WPF provides a Windows Presentation Foundation (WPF) GUI for the Windows fc.exe, file compare utility. The GUI allows the user to use a file dialog to find two files to compare. It then spawns a process running fc and displays the results in a textarea.
