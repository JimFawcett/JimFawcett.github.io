FileSizes  Repository

Used to find very large files to discard

Responsibilities:

FileSizes reports all files, matching one or more specified patterns, in the directory tree rooted at a specified path. It displays all discovered files, sorted in order of their file sizes. Users can request sorts from largest to smallest or smallest to largest. Use to find very large files and very small files that may be temporaries to discard. A small extension is planned to find the physical size of a directory and all its contents including subdirectories.

Design:

FileSizes uses a file manager to find files, storing them in a std::map, ordered by file size.

Build:

FileSizes was built with Visual Studio Community Edition - 2019, and tested on Windows 10.
