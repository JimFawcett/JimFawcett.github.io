FileDates  Repository

Used to find latest version and old discardable files

Responsibilities:

FileDates reports all files, matching one or more specified patterns, in the directory tree rooted at a specified path. It displays all discovered files, sorted in order of their file dates. Users can request sorts from newest to oldest or oldest to newest.

Design:

FileDates uses a file manager to find files, storing them in a std::map, ordered by file date.

Build:

FileDates was built with Visual Studio Community Edition - 2019, and tested on Windows 10.
