Thread Page Prototype

start a new thread, add new page to existing thread

This page is a simplification of the previous PageFramePrototype, providing all but content for a new page in a thread or the first page of a new thread. It provides top and bottom menus expected of this site's pages. Your content goes in the <content> element and provides ids and hook elements to interact with the menu controls. You can see how that is done by looking at this page's source. A thread is a linked sequence of pages that all focus on a single topic, e.g., The Rust programming language or Basic Platform information. You can see a complete list of threads supported by this site in the SiteMap. The menus contain "Prev" and "Next" buttons that support stepping through thread pages in sequence. The top menu also contains a "Pages" dropdown that displays all of the pages in the current thread, e.g., the Prototype thread. Hover to see them. Using this Thread Page prototype makes it relatively easy to add a new page to an existing thread or start a new thread. Simply make a copy of the prototype with an appropriate name and add the contents you want to display here.
