Rust TopicsDescriptions
RustTour.pdf A presentation summarizing essential features of Rust. This is a good place to start if you are new to Rust.
RustErrorHandling.pdf A presentation covering interesting mechanisms that Rust provides for handling program run-time errors.
Rust Story Narrative about the Rust Programming Language, in 8 chapters. Covers most of the details needed for productive development using Rust, with many code examples. Exercises are included at the end of most chapters.
Rust Bites Each Bite covers one Rust topic, e.g., Safety, Data Operations, Data Structures, ..., with examples and exercises at the end. Designed for those who need a quick refresher on a Rust Topic.
Rust Flash Cards Flash cards are tiny blocks of text, diagrams, and often a code expression or two. The intent is to provide a quick view of the basic ideas, perhaps accompaning study of the Rust Story or Rust Bites.
Most flash cards have a link to a code sample in the Rust Playground. There you can run code, change it, and see what happens.
Rust Glossary An alphabetic list of Rust language terms with one-line definitions and links to more information.
Rust Repositories A set of more than 20 repositories of Rust Code, all with documentation pages linked from this repository list.
Some Examples: TextFinder, Blocking Queue, ThreadPool DirNav BuildOn Project
Videos Five videos about Rust are linked from this page, as well as videos about C++ and Software Development.
C++ TopicsDescriptions
CppModels.pdf A presentation summarizing essential models for the C++ Programming Language. This is a good place to start if you are new to C++.
ClassRelationships.pdf Table of definitions, diagrams, code snippets and text.
ValueType.pdf Walk through implementation of a string class that provides copy construction and assignment, move construction and assignment, and other operations that support value behavior.
C++ Story Narrative about the C++ Programming Language, in 13 chapters. Covers most of the details needed for productive development using C++, with many code examples. Exercises are included at the end of most chapters.
C++ Repositories A set of more than 40 repositories of C++ Code, all with documentation pages linked from this repository list.
