Getting Started Resources for those who are just starting Rust.
RustTour.pdf A presentation summarizing essential features of Rust. This is a good place to start if you are new to Rust.
RustErrorHandling.pdf A presentation covering interesting mechanisms that Rust provides for handling program run-time errors.
into_rust() Page with links to five short videos by Nicholas D. Matsakis, one of the Rust team developers. These are excellent resources for starting a Rust journey.
Introducing Structs, traits, and zero-cost abstractions A YouTube video that uses simple examples for building user-defined types and abstracting out key traits. The video uses some very clever examples to get at the core of what zero-cost abstraction means.

Rust BuildOn
Repository 		BuildOn is a Process, supported by BuildOn pages, to learn rust by building a project package by package. The proecess will be conducted remotely, perhaps bi-weekly. BuildOn pages provide links to learning materials and my implementations of the previous packages. I will quicly discuss code for the last package in each remote session.
Intro, Step #0 - TF, Step #1 - Ts, Step #2 - DN, Step #3 - CP, Step #4 - Ex

Idioms and Patterns
Repository 		A sequence of pages that compare how specific idioms are implemented in C++, Rust, and C#. This may help you learn Rust by comparing how C++ or C# implements an idiom used often in Rust.
Note that you can build projects for Windows, Linux, and macOS, using Visual Studio Code IDE for all three languages, CMake for C++, cargo for Rust, and dotnet CLI for C#, and using very similar processes for each language. That is discussed in the first IaP page: Tooling.
