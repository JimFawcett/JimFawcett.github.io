WebComponents  Repository

Contains Webpage Widgets used for this website

Concept:

While creating this site, I've developed several "UI Widgets" for:
  • Resizing images
  • Displaying slide-in notes
  • Creating draggable DOM elements
  • Left and right panels sized with a splitter bar
  • Creating slide shows
  • Test loading files using JavaScript File API and ajax
  • Building simple dropdown menus - part of StoryTeller - not here
This repository is intended to store working prototypes of all the components I develop, to be used in the site as needed, and for other web applications as well. This would also be a good place to store trial apps demonstrating jQuery widgets. I haven't used jQuery in a while, but am interested to see what is in that space now.

Status:

Right now, widget tests are available for everything except dropdown menus
