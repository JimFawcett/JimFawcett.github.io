WebComponents  Repository

Contains Webpage Widgets used for this website

Concept:

While creating this site, I've developed several "UI Widgets" for:
  • Resizing images
  • Displaying slide-out notes
  • Building simple dropdown menus
  • Creating slide shows
This repository is intended to store working prototypes of all the components I develop, to be used in the site as needed, and for other web applications as well. As of now, all of the code I've developed has been classic HTML, JavaScript, and CSS. I intend to explore the W3C recommended Web Components, as well as use of React and/or Vue, for later implementations. This would also be a good place to store trial apps demonstrating jQuery widgets. I haven't used jQuery in a while, but am interested to see what is in that space now.

Status:

Right now, there is almost nothing here, but that should change soon.
