CppBlockingQueue  Repository

Thread-safe queue for transferring messages between threads

Resposibilities:

CppBlockingQueue provides a thread-safe message queue that blocks deQuer when empty. It is intended to pass messages between threads in concurrent designs, e.g., asynchronous message-passing communication systems. This is a small repository holding a single package for this reusable component.

Implementation:

The blocking queue is a template class, implemented using std::mutex and std::condition_variable.

Build:

CppBlockingQueue was built with Visual Studio Community Edition - 2019, and tested on Windows 10.

Status:

I've used CppBlockingQueue in my own designs and students have used it for several years without problems.
