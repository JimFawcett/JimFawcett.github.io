FileSystem  Repository

Responsibilities:

FileSystem-Windows is a library that supports managing directory contents on Windows. It provides File, FileInfo, Directory, and Path classes in the FileSystem namespace. This design is modeled after C# classes in the .Net System.IO namespace. Many projects included in this collection of repositories use one of the DirExplorer classes (in the FileUtilities Repository) which all use FileSystem-Windows.

Resources:

FileSystem-Windows code was built with Visual Studio Community Edition - 2019 and tested on Windows 10.
