CppUtilties Repository

CppUtilities provides helper classes and functions for Code, Strings, and Testing. It has facilities for:
  • Parsing program command lines
  • Converting between values and string representations of common types
  • Managing DateTime strings using std::chrono
  • Formatting title strings
  • Trimming leading and trailing whitespace from strings
  • Splitting delimiter separated strings and placing splits in a std::vector<string>
  • Running a sequence of code tests each wrapped in a try-catch block and announcing its results

Resources:

cppUtilties repository cppUtilities code was built with Visual Studio Community Edition, 2019. You may have to retarget the solution for the IDE version you are using.
