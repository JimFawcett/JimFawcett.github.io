MLiPS Story:

Forward
This Machine Learning process integrated with PowerShell Story pulls together pages writen by Mike Corley, presenting his ideas for a PowerShell pipeline-based architecture for managing Natural Language Processing executions.
Context
A motivating example is presented, and traditional approaches are briefly described. The concept, architecture, and implementation are introduced in subsequent pages.
MLiPS Framework  
A Framework block diagram is presented and discussed briefly. A set of stages that are typical for NLP are illustrated.
PowerShell
Background information about PowerShell is presented.
PowerShell Pipeline
This page, and the next, illustrate how processing can be dynamically configured, based on policy and objects being analyzed.
Testing Concept  
A script is presented that illustrates how the pipeline is reconfigured dynamically, based on the objects being processes. The script code shows a lot about the way this concept works.
