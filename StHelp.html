Help with Navigation:
  1. [?] button at the top right toggles visibility of page notes - commentary specific to the current story page.
  2. [Curr] button reloads the last visited story page. That is useful when you navigate away from story content using the viewed page's menu or links.
  3. [Exit] closes Story Teller and loads the site home page.
  4. [Next], [Prev], [First], and [Last] load the next, previous, first and last story pages.
  5. [Swp] toggles the ability to goto the next or previous pages using touch or mouse swipes when the focus is on StoryTeller, not its viewed content. StoryTeller acquires focus any time you click or touch anything outside the content view, e.g., the top, right, or bottom bars.
  6. One thing that may be a bit confusing: there are two kinds of navigation enabled in StoryTeller. The first and primary is navigation through story content using StoryTeller buttons and swipes. This only happens when the StoryTeller is in focus, e.g., you have clicked or touched anywhere outside the content view.
    The second kind of navigation is content navigation using content menus and navigation buttons at the lower right in the content view. This takes you away from the story sequence, but you can always return using the [Curr] StoryTeller button.
  7. The content view navigation buttons will have [N] and [P] buttons only if that content page is part of a sequence of pages, e.g., all the Blogs are part of a sequence. Sequences are defined by hidden next and prev links on the content page. If there are no such links the page is not part of a sequence, and you won't see the [N] and [P] navigation buttons.
  8. The content buttons are:
    • [S] toggles ability to swipe - appears only if page is part of a sequence
    • [R] reloads the page
    • [T] scrolls to the page top
    • [B] scrolls to the page bottom
    • [H] opens help for using content pages
    • [p] navigate to the previous sequence page - only if part of sequence
    • [N] navigate to the next sequence page - only if part of a sequence
Help with Keyboard actions - when StoryTeller is in focus:
  1. O - same as clicking [?]
  2. N - same as clicking [Next]
  3. P - same as clicking [Prev]
  4. C - same as clicking [Curr]
  5. E - same as clicking [Exit]
  6. S - same as clicking [Swp]
  7. H - same as clicking [Help]
Help with Keyboard actions - when content is in focus:
  1. S - same as clicking [S]
  2. R - same as clicking [R]
  3. T - same as clicking [T]
  4. B - same as clicking [B]
  5. H - same as clicking [H]
  6. N - same as clicking [N]
  7. P - same as clicking [P]
  8. U - scroll up
  9. D - scroll down
Help with Image Sizing:
  1. Many, but not all, images are resizable.
  2. Enlarge an image by clicking.
  3. Return to default size by refreshing the page:
    • browser refresh button
    • ^ctrl + r
    • Click on the info-bar at the bottom of the page.
    • Press the r key
    • Click the R navigation button at the bottom right of the page.
      The R button only appears if the page has resizable images.
    • If nav keys are hidden you can show them with About > Toggle Nav Keys.
    All these alternatives are here for testing. I may elect to remove some after using them for a while.
