Prologue:

This story is just beginning. I've been looking at Rust for a few days, and so know very little right now. That should change quickly, and as I learn more I will add to this story. I've started Rust Repositories. For a while there will only be demonstration code there, as I learn how Rust works. Rust is a programming language that is being developed starting from ideas from C++. However, it has many features that are uniquely its own:
  1. Data is, by default, immutable. You use the mut keyword to allow a variable to be modified.
  2. Ownership of data
  3. Traits
  4. Crates and Modules
