Rust Feature Animations

Each row of the table below, with the exception of the first, is an example of a Rust feature. The second column contains, in most cases, a link to a code example embedded in the Rust playground. The third column provides related references. The first row provides code to paste into the Rust playground to show that you can take Rust code from anywhere and run in the playground just by pasting into the playground editor. I've provided code from two sources, the first from a github Gist (small snippet of code) via a link that pops up a raw view from Github. The second is from a dropdown provided in the first row. Just cut and paste. One reason the first row is provided is to practice pasting code into the playground. Another reason is that for C++ you have to use another playground, sololearn, but you can't save code there without becoming a subscriber to the site. However, you can simply paste code into that editor and execute it, so I've provided, in the third row, code I want you to see, via Gist and embedded code (it's the same code). There is a separate RustBites_CodeExperiments Bite that explores using the Rust playground and Cargo to experiment with code.
