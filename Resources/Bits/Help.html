Bits Help

These pages are intended to help you learn a new language by comparing similar code fragments in a language you know and one you want to learn.

1.0 Bits UI Help

Each Bits page has two main sections: CodeSnaps show source code, output, and other relevant information about creating and building projects, and VS Code ScreenShots that show how to use VS Code to explore the Bits examples. The intent of these is to enable quick access to code exploration in any of the languages: C++, Rust, C#, Python, and JavaScript. There is a two dimensional grid of examples with languages on the vertical dimension and examples for a give language on the horizontal axis, e.g., Hello World, Data, Objects, ... The Bits_Repo page provides a quick summary and links to the content pages. Bits_Intro provides language and type background needed to fully appreciate information provided in the examples.

2.0 Navigation

As shown in Figure 1., Navigation within one language's examples works with the "Next" and "Prev" buttons on top or bottom menus or with "N" and "P" keys. Navigation between languages is effected by using the fixed right links panel. You can right-click on any of the links to move to that location, open in a new tab, or open in a new window. The last lets you make a direct comparison between languages if you have enough computer screen area.
