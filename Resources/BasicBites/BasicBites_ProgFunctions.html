BasicBites - Functions

Software source code is structured with functions, structs and classes, libraries, executables, and projects. Functions, the lowest of these levels, are named callable code blocks with input arguments and often a returned value.
f(args) -> returnVal { ... }

Function Types

In this section we use the term function for callable code blocks that are not attached to a type, e.g., struct or class. To make that usage clear we may refer to them as unbound or free functions.
  • Free Functions (C++ syntax)
    auto f(T t) -> T {...} /* pass by value */
    auto f(T& t) -> T {...} /* pass by C++ reference */
    auto f(T* t) -> T {...} /* pass by reference using pointer */
