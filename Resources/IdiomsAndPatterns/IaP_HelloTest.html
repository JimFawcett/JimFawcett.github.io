Hello world

Demonstration compares C#, C++, and Rust "Hello World" programs

Each item in the IdiomsAndPatterns repository focues on a simple program construct built in each of the three languages: C++ 20, Rust 1.49, and C# 9.0. This may help you use what you know of C++ or C# (or Java) to learn Rust more easily. We start here with the simplest programs in each of the three languagues. Find instructions for downloading build chains for each of these languages in Tooling.
