Contents:

This repository holds code examples used in the C++ story, ver 2:
  • Chapter1-Survey Mostly small examples used to support Chapter1 discussions:
    Elementary examples of Arrays and Pointers, STL containers, new Optional type, casts, and template function overloads.
  • Chapter1-overloading Simple overloading demonstration that is repeated using std::optional type.
  • Chapter1-overriding Uses SWDev, Dev, TeamLead, and ProjectMgr classes to demonstrate overriding.
  • Chapter1-Person Class that models a person with stats properties.
  • Chapter2-Data Demonstratons of initialization, structured binding, float granularity
  • Chapter2-sizes Uses displayType function to explore sizes of fundamental and library types.
  • Chapter2-init Illustrates uniform initialization with many small examples.
  • Chapter2-STL (uses DirWalker) Illustrates quickly building interesting data structures using the STL containers and iterators.
  • DirWalker Used std::filesystem library to build a directory navigator.
  • Chapter3-Coercion Illustrates narrowing and numeric conversions of fundamental types
  • Chapter3-CopyOperations demonstrates copy syntax and operation for fundamental types and structs
  • Chapter3-AssignmentOperations demonstrates assignment syntax and operation for fundamental types and structs
  • Chapter3-functions Illustrates functions, function pointers, methods, method pointers, functors, and std::invoke.
  • Chapter3-Logger Illustrates new logger design
  • Chapter3-STL Demonstrates use of lambda with copy_if algorithm
  • Chapter4-classes Demonstrates three variations of a point class used to discuss class-related design.
    Develops Anatomy class illustrating important class methods, e.g. Copy and move construction, copy and move assignment, destructor, etc.
  • Chapter5-ClassLayout Demonstrates how compound classes are layed out in memory.
  • Chapter5-Polymorphism Shows how virtual methods are bound to a derived class instance when invoked using a base class pointer or reference.
  • Chapter5-PeopleHierarchy A thorough walkthrough of the use and behaviors of class hierarchies.
  • CustomTraits Uses template meta-programming to define traits for specific STL containers. Those are used in display functions that are very flexible.
  • Display Uses if constexpr (...) { ... } to compile or exclude from compilation specific parts of a template class or function to build a very effective display capability.
  • Webber Very simple implementation of an application to read a source code file and change any markup to escape sequences so it can be displayed in a browser.

Build:

Built using Visual Studio Community addition - 2019, with C++17 option. Tested on Windows 10.
